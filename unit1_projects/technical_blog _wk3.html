Javascript, Why is it Popular?

From seeing the use of javascript in many websites, it is clear that there is more movement and interaction with the use of javascript in webpage design. A great website 10 things you didn't know Javascript could do gives us a taste. Flash players, mobile app designs, hangouts in webpages, screenshots. This creates a much more interactive user interface, this really helps people connect to the page using the newest technological interfaces. My favorite is that Javascript can be used to work with kinect, so you can use you're hands to manage a screen! This takes the webpage out of just being like a "paper page" and that is a wonderful tool!
